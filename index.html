Primeiros Computadores

A 1ª geração dos computadores foi marcada pelo Harvard Mark 1 (1944), Colossus (1946) e pelo ENIAC (1946)

A 2ª geração foi marcada pelos Transistores e durou de (1959 a 1965)

A 3ª geração foi marcadao pela chegada dos CI's (Circuito Integrado) que era formado por:

  • Transistor
  • Capacitores
  • Resistores

A 4ª geração foi (a partir de 1971) com os computadores pessoais de mesa e o mouse.

A 5ª geração está sendo marcada pela AI, BitCoin e MetaVerso por exemplo.

Internet, como surgiu?

Há 40 anos, enquanto os principais meios de comunicação eram o telégrafo e o telefone, os computadores eram grandes máquinas que realizavam cálculos e armazenavam informações. De forma geral, seu uso tinha fins exclusivamente científicos e governamentais.

Foi a primeira tecnologia que permitiu a transmissão de dados de forma instantânea, enquanto que o telégrafo elétrico foi o primeiro a empregar sinais elétricos para tais fins. Hoje pode parecer estranho, mas a capacidade de acelerar a transmissão de dados de forma radical não foi apreciada imediatamente.

  • Década de 1950 - Guerra Fria

Em 1958 foi criada a (DARPA) - Agência de Projetos de Pesquisa Avançada de Defesa dos militares dos EUA, em resposta ao lançamento do satélite soviético Sputnik. Seu objetivo era garantir que os Estados Unidos nunca mais ficassem para trás na corrida da tecnologia.

TCP/IP é um conjunto de regras padronizadas que permitem que os computadores se comuniquem em uma rede como a internet.

Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (www)

O que são Clients?

Cliente é todo e qualquer tipo de usuário que se utiliza da rede de internet para buscar alguma informação que está no servidor.

Responsividade

Responsividade é a capacidade que os sitem possuem de se adaptar aos variados tamanhos de tela.

Navegadores

Navegadores são progamas criados por empresas, utilizados para abrir/executar arquivos.

Aplicações Web

Aplicações Web são soluções criadas que possuem a internet como meio de comunicação entre Client x Server não sendo necessário a sua instalação.

O que são Servers?

Todo computador que é usado para se acessar seus arquivos que estão ali guardados pode ser considerado um servidor.

Teremos assim os servidores remotos e os servidores locais (LAN)

Os servidores diferem também a respeito da sua finalidade. Exemplos:

  • Arquivos
  • Segurança(Firewall)
  • Streaming
  • E-mail
  • Web

Servidores pela grande quantidade de dados que dispoem possuem hardware e software próprios projetados para seu fim específico.

DNS

O sistema DNS da internet funciona praticamente como uma agenda de telefone ao gerenciar o mapeamento entre nomes e números. Os servidores DNS convertem solicitações de nomes em endereços IP, controlando qual servidor um usuário final alcançará quando digitar um nome de domínio no navegador da web.

Sobre linguagens de programação

Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).

Linguagem Server-Side são linguagens que são executadas no servidor.

Linguagem Client-Side são linguagens que são executadas no cliente.

Afinal, HTML é linguagem de programação?

Não! HTML é uma linguagem de marcação de texto.
